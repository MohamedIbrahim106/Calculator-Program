# Calculator Program

A simple and interactive Calculator Application built using Python.  
This project demonstrates the fundamentals of programming logic, arithmetic operations, and modular design.  
It is suitable for beginners learning the basics of Python and software structure.

---

## Project Overview

This calculator allows users to perform standard arithmetic operations such as:

- Addition  
- Subtraction  
- Multiplication  
- Division  

It provides a clear structure for understanding:

- Function creation and modular coding  
- Handling user input and validation  
- Using conditionals and loops  
- Implementing error handling (for example, division by zero)
